Queryverse

Julia Packages for Data Science

Get Started

Data Manipulation

Use Query.jl to manipulate, query and reshape any kind of data in Julia.

Data Visualization

Use VegaLite.jl to produce beautiful figures using a Grammar of Graphics like API and DataVoyager.jl to interactively explore your data.

File IO

Use a uniform way to load and save CSV, Feather, Excel, SPSS, Stata, SAS and Parquet files.

UI Tools

Display plots and tables in an Electron based window with ElectronDisplay.jl
